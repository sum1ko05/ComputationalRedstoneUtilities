# ComputationalRedstoneUtilities
Just programms that could assist in compiling/running something on redstone computers in Minecraft

## Getting started
To get this project you should just `git clone` this repository, and than build this project with CMake. You should get the executable file at this point. To make the example binary you should launch this executable in terminal and pass to it sample assembly source from sample_programs/

## Available resources
* REM-1 (Redstone Experimental Machine 1)
    * Instruction reference - finished, but not suitable for public access
    * Assembler - main functionality finished
    * B2Sch (Binary to schematic) - not started yet
